# Spring Boot - Spring MVC Validation - Strings for Int Fields and Custom Messages

Hands On: `06-spring-mvc/code/02-validationdemo`

## Development Process

1. Create custom error message
   - `src/resources/messages.properties`

```
typeMismatch.customer.freePasses=Invalid number

# Error Type: typeMismatch
# Spring model attribute name: customer
# Fields name: freePasses
# Our custom message: Invalid number
```
